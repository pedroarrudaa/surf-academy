About This Project

SurfAcademy is a Next.js application that serves as a platform for WindSurf IDE tutorials and blogs. It features a responsive design that works on both mobile and desktop devices.

The application includes features such as:

  • Browse WindSurf IDE tutorial videos
  • Read blog posts about WindSurf IDE and Codium AI integration
  • Video transcription and analysis using AssemblyAI
  • Intelligent chapter generation from video content
  • Dark mode support

Note: This GitHub Pages deployment is a static version with limited functionality. For the full experience with all features working, please run the project locally.

Run Locally

To run this project locally and experience all its features:

  1. Clone the repository:
    git clone https://github.com/pedroarrudaa/surf-academy.git cd surf_academy
  2. Install dependencies:
    npm install
  3. Start the development server:
    npm run dev
  4. Open http://localhost:3000 in your browser.
